Java Go


For Hassle-free coffee runs

With Java Go, coffee runs are faster, more efficient and stress-free

Java Go connects you to great independent cafés across London!

Features of the App

  • Allows you to invite colleagues to a coffee run (no need to ask them individually) - which removes the need to write orders or memorise them - No need to ask colleagues to pay you back
  • Lets you discover the best independent cafés in your area
  • Allows you to search cafés using specific categories (i.e. vegan cafés)
  • Automatically stamps your loyalty card
